# Skill Assessment Guide

- "What is a variable?"

- "What is a function?"

**Intermediate indicators:**
- "Explain closures"
- "What is the event loop?"
- "Describe async/await vs Promises"

**Advanced indicators:**
- "Explain prototypal inheritance"
- "What are generators and when to use them?"
- "Describe memory management in JS"

## Skill Level Definitions

### Level 1: Beginner

**Characteristics:**
- New to programming or specific technology
- Needs fundamental concepts explained
- Requires step-by-step guidance
- May struggle with syntax

**Teaching approach:**
- Explain every line of code
- Use simple analogies
- Avoid jargon or define all terms
- Smaller, more frequent sessions
- More verification checkpoints

### Level 2: Intermediate

**Characteristics:**
- Understands basics
- Can read code but struggles writing from scratch
- Knows some patterns but not when to apply them
- May have gaps in foundational knowledge

**Teaching approach:**
- Explain the "why" more than the "what"
- Introduce design patterns
- Challenge with edge cases
- Encourage problem-solving before showing solutions

### Level 3: Advanced

**Characteristics:**
- Comfortable writing code
- Understands common patterns
- Needs guidance on best practices
- Learning optimization and architecture

**Teaching approach:**
- Focus on trade-offs and alternatives
- Discuss architecture decisions
- Introduce advanced patterns
- Code review style feedback

## Adaptive Assessment

### During Sessions

Watch for signals:

**Struggling indicators:**
- Repeated questions on same concept
- Errors in basic syntax
- Confusion about terminology
- Long pauses before responding

**Action:** Slow down, revisit fundamentals, add more examples

**Progressing well indicators:**
- Asks about edge cases
- Suggests alternatives
- Catches potential issues
- Makes connections to other concepts

**Action:** Increase complexity, introduce advanced topics

### Re-assessment Triggers

Re-evaluate skill level when:
- Introducing new technology stack component
- Learner expresses confusion repeatedly
- Learner asks for more challenge
- Starting a new project phase
